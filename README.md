# 🍞 도라에몽의 핑계빵

![cover](./animal_league/images/cover.png)

> 시험기간 대학생들의 현실적인 핑계를 담은 웹 타자 게임  
> 공부는 못 했어도, 핑계만큼은 완벽하게.

## 📌 프로젝트 소개
도라에몽의 핑계빵은 시험기간 대학생들이 공감할 만한  
‘공부를 미루는 핑계’를 소재로 한 웹 타자 게임입니다.  
제한 시간 안에 문장을 정확히 입력해 점수를 얻고,  
결과와 랭킹을 확인할 수 있도록 구현했습니다.

## ✨ 주요 기능
- 문장 타자 게임
- 점수 및 랭킹 확인
- 시작 / 결과 화면 구성

## 🛠 기술 스택
- Frontend: HTML, CSS, JavaScript
- Backend: Python(Flask), MongoDB
